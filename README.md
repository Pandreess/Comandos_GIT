ELABORADO POR: ANDRES FELIPE JARA ZARAZA
El proyecto “Comandos Esenciales” es una recopilación práctica y actualizada de los comandos más importantes utilizados en Git, una de las herramientas fundamentales para el control de versiones y trabajo colaborativo en proyectos de desarrollo de software.

Este repositorio fue creado con el objetivo de servir como una guía rápida, clara y funcional tanto para principiantes que están aprendiendo Git por primera vez, como para desarrolladores experimentados que necesitan una referencia de comandos esenciales en su flujo de trabajo diario.

La estructura del documento está organizada por secciones, cubriendo desde la configuración inicial del usuario, hasta tareas más avanzadas como el manejo de ramas, repositorios remotos, restauración de cambios, etiquetas y control de versiones.
Cada comando incluye una breve descripción de su función, facilitando la comprensión y el aprendizaje práctico.
